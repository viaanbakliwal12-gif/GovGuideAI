# GovGuideAI

GovGuideAI is an AI-powered assistant that helps Indian citizens understand government schemes, services, passport processes, required documents, and grievance procedures.

It provides simple, personalized guidance based on user details such as age, state, occupation, and other relevant profile information.

## Main features

- Personalized government scheme guidance
- Information from official Indian government sources
- User profile system
- AI chat assistant
- Scheme search tool
- Word count tool
- Simple and organized frontend and backend structure

## Project structure

- `app/` – backend logic, agents, tools, authentication, and profiles
- `templates/` – HTML pages
- `static/` – CSS and JavaScript
- `data/` – scheme data
- `docs/` – project documentation
- `main.py` – main application file
